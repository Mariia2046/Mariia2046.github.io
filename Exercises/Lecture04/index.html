Відповідь на питання про кольори:

Шістнадцятковий код:

  • чорний: #000000
  • синій: #0000FF
  • зелений: #008000
  • червоний: #FF0000
  • білий: #FFFFFF
  • жовтий: #FFFF00

RGB

  • чорний: rgb(0, 0, 0)
  • синій: rgb(0, 0, 255)
  • зелений: rgb(0, 128, 0)
  • червоний: rgb(255, 0, 0)
  • білий: rgb(255, 255, 255)
  • жовтий: rgb(255, 255, 0)

Відсоткове представлення:

  • чорний: rgb(0%, 0%, 0%)
  • синій: rgb(0%, 0%, 100%)
  • зелений: rgb(0%, 50%, 0%)
  • червоний: rgb(100%, 0%, 0%)
  • білий: rgb(100%, 100%, 100%)
  • жовтий: rgb(100%, 100%, 0%)

Використання альфа-каналу (задання прозорості):

Для всіх кольорів я можу додати альфа-канал у форматі rgba, де альфа варіюється від 0 (повна прозорість) до 1 (повна непрозорість). Наприклад, якщо я хочу зробити колір чорний з прозорістю 50%, я можу використати rgba(0, 0, 0, 0.5).
